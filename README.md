# Workout Community Portal

A modern community platform for street workout athletes.

## About

Workout Community Portal is an open-source project designed to unite the street workout community after the archival transition of workout.su.

The project focuses on:

- Personal workout tracking
- Training programs
- Outdoor workout spot map
- Community events
- QR-based attendance
- Community achievements
- Community rank based on contribution

## Project Status

Currently in active MVP development.
